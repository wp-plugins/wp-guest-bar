== Description ==
Adds a BuddyPress guest bar to your WordPress site with link to log-in or register in your website.
You can also set a custom message to invite users.
Fully localized front-end!

An italian demo can be found at [wpgov.it](http://wpgov.it)

Useful with a BBPress installation or just if you want to engage new and old users to log into your website.

== Installation ==
1. Upload the entire folder to the '/wp-content/plugins/' directory or install direcly via the WordPress plugin screen
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy!
