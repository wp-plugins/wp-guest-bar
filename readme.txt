== Description ==
Adds a BuddyPress-like guest bar to your WordPress site, giving guest users a link to log-in or register in your website.

An italian demo can be found at [wpgov.it](http://wpgov.it)

This could be useful, for example, with a BBPress installation or just if you want to engage new and old users to log into your website.

WP Guest Bar is a new plugin. More features and customization will be added soon...

If you like WP Guest

== Installation ==
1. Upload the entire folder to the '/wp-content/plugins/' directory or install direcly via the WordPress plugin screen
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy!
